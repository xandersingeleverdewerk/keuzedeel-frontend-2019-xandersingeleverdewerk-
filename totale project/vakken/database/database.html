Database

Een database is een gegevensbank waarin de gegevens van verschillende soorten worden opgeslagen. De kennis over Databases is belangrijk omdat elk bedrijf er iets mee doet. Zo moet je o.a. databases kunnen bouwen, gegevens uit de databases halen en de gegevens aan kunnen passen of verwijderen. Dat alles gebeurt door middel van SQL. Omdat over SQL veel verteld kan worden hebben we besloten om op deze webpagina slechts de basis uit te leggen en om verdere informatie te vertellen door middel van de tutorials die via deze webpagina bereikt kunnen worden. Zoals al verteld kan er van SQL gebruik worden gemaakt om o.a. gegevens aan te passen, gegevens te verwijderen en om gegevens toe te voegen. Dat wordt gedaan door middel van codes. Bij die codes kunnnen er ook attributen worden neergezet die in de database moeten staan. Je kan er snel achter komen om de attributen door o.a een uml op te bouwen. U kunt door middel van de tutorials zien hoe zoiets gemaakt moet worden. Omdat de klanten natuurlijk zeker willen weten dat de werknemers hun begepen hebben zullen er ook verslagen moeten worden gemaakt. Om die reden is het noodzakkelijk om bij de examens te weten hoe je zoiets kan maken. De reden waarom ik dat op dit pagina vermeld is dat de databases verbonden zijn aan die verslagen. Hoe u zo iets moet maken kunt u zien op de tutorials. We wensen u heel veel succes met het voorbereiden voor de examens.
