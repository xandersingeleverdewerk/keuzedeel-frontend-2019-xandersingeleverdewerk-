Frontend

Op dit pagina kunt u lezen over wat u over frontend moet weten. Frontend zijn de procesen die zichtbaar worden op bijvoorbeeld een website. Omdat er heeveel soorten talen voor frontend bestaan en het enigste wat u voor de examens moet kunnen is het kunnen produceren en opleveren van een werkende applicatie, hebben we besloten om het maar over de 3 basis talen te houden, namelijk html Css en javascript. Html is een script taal die bedoeld is om een website op te maken. Omdat een website eerst opgemaakt moet worden kan er gezegd worden dat html de basis taal is voor websites. Omdat echter een basis website tegenwoordig niet goed genoeg meer is hebben we besloten om ook over CSS te houden. CSS is de taal die gebruikt wordt om de website vorm te geven. Op dit manier komt de website er beter uit te zien. Omdat de CSS codes pas geschreven moet worden wanneer de html codes af zijn kan er gezegd worden dat de CSS de uitbreiding is op html. Net zoals Javascript kan er door middel van css animties op de website gemaakt worden. Omdat CSS er echter niet voor kan zorgen dat op websites complexe procesen gebeuren hebben we toch besloten om het over javascript te hebben. Javascript wordt echter in tegenstelling tot html en CSS niet altijd gebruikt omdat niet alles interactief moet zijn om het te doen op een website. Daardoor kan er gezegd worden dat javascript een niet verplichte uitbreiding is op html en CSS. Veel succes met oefenen
